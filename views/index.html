Barba.js Page Transitions 101

This is a project to explore and experiment with how barba.js works. There are many examples of how to use this page transition library. Each git branch has a different transition style with slightly different page layout styles.

This is a self-learning project to better understand how to use page transitions in real life projects. I hope you like it.
