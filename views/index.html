Barba.js Page Transitions 101

This is a project to explore and experiment with how barba.js works. This is a high concept overview to get you started, it's the result of my own learning. If you want a full in-depth review, try the docs. This guide will give you a practical understanding of how to implement barba.js for page transitions for your next project.

Animations

Before learning how to setup and use animations in your code, take a step back and consider when and where to implement them. Think about how they interplay with UX. Imagine all the possibilities of how they can fit into your interface designs and create eye-catching microinteractions.

This is a self-learning project to better understand how to use page transitions in real life projects. I hope you like it.

How to Use

This guide is arranged with git branches, because that is how I put this together. You will need to understand how to checkout git branches. As you checkout each branch you can see the code in your editor to follow along. Run npm run server to start up a local dev server in your browser. Make changes to the code, save the file and see the changes in your browser!

  1. Dowload or Clone from repo.
  2. Run npm install in your terminal to install dependencies.
  3. Run npm run server to start a local dev server.
