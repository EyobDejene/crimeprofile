Features

Barba.js comes with several plugins for additional functionality.

CORE

The core of barba.js, you need this to get page transitions working.

CSS

Use css animations & transitions to power page transitions. Useful if you don't know javascript.

ROUTER

Setup transitions for your page routes.

PREFETCH

Prefetch your pages as the user hovers over a link before they click.
